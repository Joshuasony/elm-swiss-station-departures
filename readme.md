# elm-swiss-station-departures

A simple app to look up the next departures from a given station

## Installation and running

Install elm (if not already installed).


```
npm install -g elm
```

Verify Installation:
```
elm --version // => 0.17.1
```

Clone this repo.

```
npm install
npm run dev
```

Compiles and runs to app on http://localhost:3000
